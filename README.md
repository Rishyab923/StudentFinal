# Student Registration System

A full-stack web application for student registration built with React and Node.js.

## Setup Instructions

### Backend Setup
```bash
cd backend
npm install
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## Features
- Student registration
- Course listing
- Responsive design
- MongoDB integration

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB
